Avantages :
  • C'est mode est écologique car l'image n'est stockée ni dans la boîte d'envoi, ni dans la boîte de réception du destinataire.
  • Ce mode utilise SVG ou PNG en fonction de la compatibilité du logiciel mail.
Inconvénients :
  • Si le client de messagerie est configuré pour ne pas télécharger les images, le logo ne sera pas visible (le reste de la signature si).
  • Il est possible qu'avec des clients mails très anciens ça ne fonctionne pas bien.
Avantages :
  • Le logo est toujours visible par le destinataire (sauf s'il a interdit les messages en html).
  • L'image PNG est d'une petite taille.
Inconvénients :
  • Ce mode n'est pas très écologique car l'image est stockée dans la boîte d'envoi et dans la boîte de réception du destinataire.
  • La qualité de l'image PNG n'est pas aussi bonne que celle de l'image SVG.
